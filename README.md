# Sainsbury Serverside Test
Build a Java console application that scrapes the Sainsbury’s grocery site’s - Berries, Cherries, Currants page and returns a JSON array of all the products on the page.

## Dependencies
* spring - Spring Framework 5.0.6
* apache common - Apache Common library 3.9.0
* jsoup - HTML Parser 1.11.3
* gson - Serialise Java Objects into JSON 2.8.5 
* lombok - Boilerplate code annotations 1.18.6
* junit - Unit testing framework 4.12
* easymock - Easymock testing framework 4.0.2

## Build & Run with Maven
* Build the project
```
mvn clean install
```
* Run the application
```
mvn exec:java
```

## Build & Run from Jar:
* Build the project
```
mvn clean package
```
* Run the application
```
java -jar target/ServersideTest-1.0.0-jar-with-dependencies.jar
```

Run from your Java IDE:
* Clone the project on your Java IDE and run the application

## How to run tests
Run with maven:
```
mvn clean test
```
